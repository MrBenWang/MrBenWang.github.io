*在MacBook Air 上装Win10的，反反复复的失败过程。

2015-08-10 18:19
生活随笔
约 2192 字 预计阅读 5 分钟

文章目录

这个月初，一个女性朋友托我帮她装电脑，往 MacBook Air上面装 Windows 系统，原因是windows用的习惯，用起来顺手。
然后用脚趾头考虑了一下，就一口答应下来了。难道这就是一个标准程序员的命么，修电脑，装系统，最近又多了一个任务，帮忙买手机。o(╯□╰)o

在家里，我先简单百度了一下Mac是什么鬼，然后顺便看了一下网上有一大堆苹果电脑装Windows 的教程，可见还是有很多的成功例子。也没就没有仔细去看，然后就放心大胆的在家里，装备个U盘，用大白菜做成U盘启动。顺便也幻想了一下，顺利的装机后，请我吃饭的场景了。可是，结果完全不是这个样子的。

一、我到她家后，先询问了一下：这里面有没有什么需要保持备份的。得到“没有”这个痛快的回答后，我就插上U盘，着手装系统了。首先发现苹果电脑的键盘怎么是这个样子的，完全跟我想象的不一样，BIOS也不一样！这怎么调用U盘启动啊！ 还好有百度，不能慌。然后淡定的找到Option按钮，就是选择启动盘的。进入WinPE后，按照我以往的经验，先格式化硬盘，再重新分区，再安装系统。 一切完毕后，重启。觉得，这也太简单就赚到一顿晚餐了，好轻松，哈哈。

可是重启过后，我傻眼了，系统提示，找不到启动设备的（英文大概是这个意思，具体是什么记不清了）。然后就是不停的疯狂百度，找原因，找解决方法。在朋友面前这么sui，面子过不去，就也不仔细看看网上是怎么解释的，只想尽快的解决问题，按照解决方案一个个的试。Command+Option+R+P、听到“duang”响三声后松开，或者听到“duang”响四声后松开，或者联网自动修复什么的。 总之是各种方法都试过了，然而并没有什么用，还得给她解释“我没怎么玩过苹果”“这个错误太奇怪了“”网上怎么也找不到啊”。这时候女神发话了，要不你拿回去慢慢弄吧。 然后赶紧答应了下来，修好后，立马给你送过来。

二、回家后，心就冷静下来了，慢慢查资料，分析原因。一个个的试错。最后皇天不负有心人，终于找到原因了，是MBR出问题，当时在格式化后，没有对这个重建。导致系统找不到硬盘。大学时候学的知识真是忘得一干二净了。。。。重新进入后，找到硬盘顺利进入了win10的第一次设置界面，设置时区，设置语言，下一步下一步，当点了快速设置后，duang~~一声，重启了。嗯，也对，第一次设置得重启几次得样子。

可是跟我想象的不一样啊！又重新进入了设置界面，无论重启多少次。我有仔细回想了一下，可能是我打开方式不对，当时不是硬盘MBR重建了么，可能是这个原因。 紧接着开始第二次重装，进去后，又是老问题。会不会是我装的win10家庭版的问题呢？这个问题立马被我否决了，因为这个算是最小的版本了。

然后又不停的百度，搜索，发现可能是苹果电脑的硬件兼容问题。无意间还发现，windows有个审核模式，苹果电脑进入方式是FN+shift+command+F4(可能还是F2，记不太清了)。进来后直接就是 administrator 登陆的 win10系统，然后我就给它下载驱动精灵装驱动，然后重启等待，还是老样子。 然后又进入审核模式，删了驱动，重新进去还是老样子。

无奈了，看到论坛上面说，windows 必须在 macOS 系统中用 bootcamp，不能用ghost，并且苹果系统没有BIOS，并且苹果的硬盘格式跟windows也不一样。。。。我也是醉了，高大上的 Mac非要和屌丝 windows区分的这么明显么。。。

然后就是按照上面说的，先重装Mac OS X，因为Mac坏了，只能在 windows里做个 U盘装，找来找去找到这个软件TransMac，制作苹果系统U盘的。装好后，在系统里面做 windows的U盘。这时候我才发现，当时我想把两个系统分在不同的盘里，这里根本不行，必须在是一个盘。我不得已，又重装了一遍，划分整个硬盘（只有可怜的60G）给它了。 然后就按照教程，和提示一步一步的走下去。当我再次重启后，终于看到了喜人的win10登录界面。然后如释重负，突然轻松了，突然睡意也袭来了。

此时凌晨2点半。折腾了这么久，也不想折腾了。所以什么软件都没给它装了。干干净净挺好。

如上图的，用了三个U盘，和一台笔记本，才搞定得了这个破电脑，

  • 第一个金士顿的16G（回家路上捡的，里面没有爆料），用做 mac下的 windows U盘启动。
  • 第二个爱国者2G（大学时候省吃俭用买下的，当时2G很大了），用做winPE，用来修复硬盘的MBR。
  • 第三个闪迪8G（买键盘送的），里面装的是 重装Mac OS X系统的U盘。

后记：后来当我再用U盘 copy的时候，才发现Window识别不了，又去网上找了一堆修复工具，一个都没用处。最后还是在windows的设备里面 把它重新格式化了。。。

总结一下：

  1. 装电脑这件事，自以为是的认为都是一样的，可是当他真正的去装的时候，就大不一样。
  2. 遇到问题不能太浮躁，一定要冷静，静下心来。（这句话都老生常淡了，还是要再提醒一下）
  3. 做事之前，一定要预先看一些资料，预习的重要性可以少走很多冤枉路。（可是，现实中很多高人都是自己走了太多的冤枉路，才能修炼成高人的。看个人理解了）

一定要提醒注意事项，即使99%能成功的事情，也要做好失败的准备。（这次对方幸好没有重要的东西，要不全部搞丢了，我就难堪了）

看了各位的评论，真是哭笑不得，也就不一一回复了。

如果说，对对方没想法那是假的，可是这 该怎么区分：对方是真的需要装系统，还是…。求大神指点一二。
